# Gîte Master v2.0

Application de gestion de locations saisonnières.

## Démarrage rapide

```bash
# Installer les dépendances
npm install

# Lancer en développement
npm run dev

# Builder pour production
npm run build
```

## Déploiement Vercel

Ce projet est prêt pour être déployé sur Vercel.

1. Push sur GitHub
2. Import dans Vercel
3. Deploy automatique

## Modules

- Dashboard
- Calendrier réservations
- Messagerie clients
- Comptabilité (taxe de séjour auto)
- Synchronisation multi-canaux
- Administration

## Technologies

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
